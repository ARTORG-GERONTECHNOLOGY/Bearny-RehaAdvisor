## Canonical Production Docs

1. `06-DEPLOYMENT_GUIDE.md`

2. `PRODUCTION_DEPLOY_RUNBOOK.md`

1. Read `06-DEPLOYMENT_GUIDE.md` for architecture and prerequisites.
2. Use `PRODUCTION_DEPLOY_RUNBOOK.md` for the exact release-to-production procedure.
3. Use `08-TROUBLESHOOTING.md` for failure diagnosis.
